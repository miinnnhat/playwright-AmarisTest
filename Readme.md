# Playwright Amaris Test Automation Dang Minh Nhat

This repository contains an automated test project built with **Playwright** using **JavaScript**, created as part of a technical test / study project for **Amaris**.

The main purpose of this project is to demonstrate:
- Web UI test automation using Playwright
- Clean test structure and configuration
- Basic CI-ready setup
- Best practices for modern end-to-end testing

---

## 🧰 Tech Stack

- **Node.js**
- **Playwright**
- **JavaScript**
- **GitHub Actions** (CI workflow)
